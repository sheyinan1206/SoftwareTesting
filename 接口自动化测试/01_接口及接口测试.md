## 1.接口是什么

### 概念

两个独立系统或组件进行**交互**和通信的边界点或连接点定义了**交互**的规范而不涉及内部具体实现从而让不同实体之间能够协同工作

### 意义

解耦，复用，标准化，生态

### 示例

- 硬件接口：USB
- 系统接口：Win32 API
- **软件接口：API** ———— 浏览器和APP协调工作

API 应用编程接口：各个应用（程序）进行高效沟通和合作

## 2.为什么测试接口

### 测试金字塔：UI API UNIT

测试的类型：

- Web、App：UI层
- **API：集成层**（承上启下、投入产出较高）
- 单元测试：白盒测试，由开发进行

### 接口测试的特点

- 效率：UI一个用例十几秒，API一个用例0.3秒
- 收益：提前介入，快速评估，避免全平台BUG，敏捷开发-测试驱动开发
- 生态：非常成熟

## 3.重点：HTTP接口

基于HTTP协议的接口，简称HTTP接口

### HTTP接口的应用

当今**互联网**应用最广泛的接口————90%以上的服务

### 基本特点

- 交互简单：请求->响应模式，无状态
- 结构简单：请求->响应，都是：行、头、体
- 内容简单：除了体，其他均为ASCII内容
- 应用广泛：至于体，支持JSON、HTML、XML、图片、视频、音乐
- 跨平台兼容：电脑、电视、手机、智能家居

1. 请求（request）
  - 行：方法
  - 头：描述（修饰）体
    - 缓存：Cache-Control、Last-Modified
    - 身份：User-Agent、Authorization
    - 内容：Content-Type、Content-Length
  - 体：无限可能

> GET和POST区别是什么？

  安全：加密 HTTPS SSL

  附录1：**常用**的HTTP方法

  

|方法名|安全性|幂等性|可缓存性|应用场景|
|:--|:--:|:--:|:--:|:--|
GET|√|√|√|获取数据、下载资源|
HEAD|√|√|√|与GET相同，但是没有正文|
POST|×|×|×|提交数据、创建资源|
PUT|×|√|×|修改数据、覆盖资源|
PATCH|×|√|×|修改【部分】数据|
DELET|×|√|×|删除数据、资源|
CONNECT|×|×|×|创建代理隧道|
OPTIONS|√|√|×|显示通信选项|
  
2. 响应（response）
  - 行：状态码
  - 头：描述（修饰）体
    - 缓存：Cache-Control、Last-Modified
    - 内容：Content-Type、Content-Length
  - 体：无限可能

  附录2：**常用**的状态码

| 状态码 | 英文名称 | 含义说明 |
|--------|----------|----------|
| 100 | Continue | 服务器已接收请求头部，客户端可继续发送请求体 |
| 101 | Switching Protocols | 协议切换，如 HTTP 升级为 WebSocket |
| 200 | OK | 请求处理成功，正常返回数据 |
| 201 | Created | 资源创建成功（POST 新增接口） |
| 204 | No Content | 请求成功，无返回内容（删除接口常用） |
| 206 | Partial Content | 分片请求，返回部分资源（断点续传） |
| 301 | Moved Permanently | 永久重定向，资源永久更换地址 |
| 302 | Found | 临时重定向 |
| 304 | Not Modified | 资源未更新，使用浏览器本地缓存 |
| 307 | Temporary Redirect | 临时重定向，保持原请求方法 |
| 400 | Bad Request | 请求参数、格式错误 |
| 401 | Unauthorized | 未登录/缺少 token 身份凭证 |
| 403 | Forbidden | 已登录，但无访问权限 |
| 404 | Not Found | 接口/页面资源不存在 |
| 405 | Method Not Allowed | 请求方式不匹配（如接口只支持 POST） |
| 408 | Request Timeout | 请求超时 |
| 409 | Conflict | 资源冲突，重复创建数据 |
| 413 | Payload Too Large | 上传请求体过大 |
| 415 | Unsupported Media Type | 不支持提交的数据格式（非 JSON） |
| 500 | Internal Server Error | 服务器内部代码异常 |
| 502 | Bad Gateway | 网关错误，后端服务返回无效响应 |
| 503 | Service Unavailable | 服务不可用、宕机、维护、限流 |
| 504 | Gateway Timeout | 网关调用后端超时无响应 |
| 505 | HTTP Version Not Supported | 不支持当前 HTTP 版本 |
