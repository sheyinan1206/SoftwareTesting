# pytest基础
1. 环境准备
2. 安装pytest 

    ```pip install pytest```
   
3. 创建用例
    - 文件：test_开头
    - 函数：test_开头
    - 内容：拥有断言
    - 返回值：必须是none
  
4. 执行用例
    - pytest命令（推荐）
    - pytest启动函数（main）
    - IDE按钮（随着编辑器不同结果也不同--不推荐）
