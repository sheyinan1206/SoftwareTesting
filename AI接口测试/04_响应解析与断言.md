## 什么是断言

断言：判断实际测试结果和预期结果是否一致的检查点，不一致则标记为失败

## Apifox常用的断言类型

- 状态码断言（2xx、4xx、5xx）
- 响应体断言

## 使用JSONPath进行响应体断言
```json
{
    "code": 0,
    "data": {
        "userName": "张三",
        "orders": [
            {
                "id": 10086
            },
            {
                "id": 1
            }
        ]
    }
}
```
- $表示根节点
- .用于访问子节点（$.data.name）
- []用于访问数组元素
- *匹配所有元素（$.data. * 匹配data下所有子节点）
